# Accademia-2.0

To build this module install [nasher](https://github.com/squattingmonk/nasher)

Then, run nasher-installer.bat, it will build a module inside .bin/modules folder from the source files in src folder

Once you have the module built, you can open it in the toolset by running toolset.bat

If any changes are made to the module from the toolset, remember to save and run nasher-unpack.bat to generate new source files from the module. Git should detected only changed files.
